# Ada Carbon Monitoring - Handover Presentation

Slidev presentation for the Ada Carbon Monitoring handover.

**View the presentation:** https://prodmodfour.github.io/ada-carbon-monitoring-presentation/

## Running the Presentation

```bash
# Install dependencies
npm install

# Start dev server (opens in browser)
npm run dev

# Build static site
npm run build

# Export to PDF
npm run export
```

## Files

- `slides.md` - Presentation content
- `package.json` - Dependencies and scripts

## Viewing

After running `npm run dev`, open http://localhost:3030 in your browser.

Navigation:
- Arrow keys or Space to advance
- Press `o` for slide overview
- Press `f` for fullscreen
